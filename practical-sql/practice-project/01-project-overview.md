# Practice Project: Personal Reading Log

A hands-on project to reinforce SQL fundamentals from Chapters 1–11. Estimated time: **1–2 hours**.

## Objective

Build a small database to track your book collection and reading history, then run queries that apply the key concepts from the first 11 chapters.

## Concepts Applied

| Chapter | Concepts Used |
|---------|---------------|
| 1 | CREATE DATABASE, CREATE TABLE, INSERT |
| 2 | SELECT, WHERE, ORDER BY, DISTINCT, LIKE |
| 3 | Data types (varchar, integer, numeric, date) |
| 4 | COPY (optional: export results) |
| 5 | Math, percentages, `percentile_cont()` |
| 6 | JOIN, NULL handling |
| 7 | Primary/Foreign keys, constraints |
| 8 | GROUP BY, HAVING, aggregates |
| 9 | ALTER TABLE, UPDATE, backup |
| 10 | `rank()`, rates |
| 11 | `date_part()`, intervals |

## Deliverables

1. Database with 2–3 tables
2. Sample data (10–15 rows)
3. 8 guided query tasks
4. 1 optional challenge

## Prerequisites

- PostgreSQL installed and running
- pgAdmin or `psql` access
- Completed Chapters 1–11 (or equivalent)
